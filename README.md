# -triply-

Triply is your all-in-one app for afforadble and authentic travel. Triply will prompt search for the cheapest option between flights with different airlines (Delta, United, American, etc), transportation with different ride share apps (Uber, Lyft, Bolt, Waymo, etc) to find the cheapest options and tailor it to the unique cultural recommendation that the app builds from your user imput and create a unique user profile.
